Tags para Texto

Agora vamos conhecer as principais tags que se aplicam aos textos das páginas Web.

Os Títulos (h1 até h6)

Para criar os títulos que aparecem em destaque nas páginas, no início de um texto e de uma seção, são utilizadas as tags h1 a h6, sendo a primeira a de maior tamanho, com as demais - h2, h3, h4, h5 e h6 - reduzindo o texto dos títulos cada vez mais.

Por convenção, o título principal da página é sempre o h1, comportando apenas um por página. Os demais títulos variam entre o h2 e o h6. Um exemplo, para uma noção dos tamanhos, o título desta página "Tags do Body" é um h1, enquanto o título desta seção - "Tags para Texto - é um h2, e o título acima "Os Títulos (h1 até h6)" é um h3. Porém, essa questão de tamanho é muito relativa, pois após o HTML ser finalizado, as páginas recebem uma estilização com o CSS (cenas de um próximo capítulo...), que poderá aumentar ou diminuir o tamanho, conforme o necessário.

Tags para Links

Link significa ligação, ligar uma coisa com outra. Aqui os links são utilizados para ligar uma página com outra página ou mesmo um local da página com outro na mesma página.

Links para outras páginas externas

Para ligarmos nossa página com outras páginas, podemos disponibilizar um link que, ao ser clicado pelo usuário, irá redirecioná-lo para outra página. Para isso temos a tag "a" para ancoragem dos links. Essa tag é especial com características próprias, como poderá ver a seguir.

Na imagem acima estaria disponível um link para a página da DIO. Assim, quando o usuário clicasse, abriria uma nova aba com a página da DIO. Como podemos ver, a tag "a" tem alguns atributos especiais. O atributo "href" é onde informamos o endereço da página para qual o usuário será direcionado. O atributo "target" é muito importante, pois ele diz se a página de destino será aberta na mesma página ou em uma nova aba. Se o atributo target não estiver presente, então será aberta a página do link na mesma aba, e a página anterior (no caso, a sua) será fechada. Normalmente, queremos que a pessoa permaneça em nossa página, servindo o link apenas para uma consulta externa. Então, se recomenda o uso do atributo target com o valor "_blanck", como demonstrado na imagem acima, assim será aberta uma página numa nova aba, mantendo a anterior aberta. O atributo "title" (não confundir com a tag title) serve para informar um texto que será exibido numa caixinha no momento que o usuário passar o ponteiro do mouse sobre o link.

Note na imagem que todos os atributos estão na tag de abertura da tag "a", e que logo em seguida, antes da tag de fechamento, temos um texto: Dio.me. Esse texto é o que vai aparecer para o usuário, sendo ele mesmo o próprio link para ser clicado.

Em primeiro lugar temos que criar uma nova página html dentro do mesmo projeto, junto com a página principal index.html (por convenção, sempre é utilizado esse nome para a página principal). Com a nova página pronta, podemos incluir um link na página principal.

Tags para Imagens

Por último, mas não menos importante, temos as tags para imagens. Para deixar as páginas mais bonitas e para ilustrar as informações trazidas nos textos é muito comum o uso de imagens. A imagem pode estar salva em uma pasta no projeto ou vir de algum outro site que disponibiliza na própria Web gratuitamente, sem direitos autoriais (muito importante verificar isso antes de publicar uma imagem em sua página). Veja como é fácil disponibilizar uma imagem na página:

No primeiro caso, a imagem está sendo buscada num site da Web. Isso mesmo, podemos utilizar imagens de outras páginas, desde que elas não estejam protegidas por direitos autorais. Então, basta utilizar a tag img, com o atributo src, que é onde será informado o endereço da imagem (o que aparece na barra de endereço, é só copiar e colar aqui). Outro atributo que é utilizado é o "alt", caso a imagem não carregue, aparece a mensagem de texto ao invés de ficar um vazio ou uma mensagem de erro. E o atributo "width", que serve para informar a largura, definindo o tamanho que a imagem será exibida na página.

No segundo caso, a imagem está salva em uma pasta dentro do projeto. É muito comum nos projetos essa pasta para guardar as imagens que serão exibidas na página. Assim, basta informar na "tag img", no atributo "src", a pasta e o nome do arquivo da imagem. Os atributos "alt", para o caso de a imagem não carregar, e o "width" para determinar o tamanho da imagem na página.
