Atributos de Estilo

Agora vamos conhecer um pouco sobre a estilização em HTML, demonstrando que ela é possível, porém não é muito usual em projetos maiores, pois, para essa finalidade existe o CSS (Cascading Style Sheets - Folhas de Estilo em Cascata). O CSS tem suas características e linguagem própria, e por isso é trabalhado num arquivo separado dentro do projeto. Como mencionado no início, o HTML é utilizado para dar a estrutura da página, e a parte de dar estilo não faz parte do seu foco, mas pode ser utilizado para algumas partes específicas. Isso pode ser feito de duas maneiras, como veremos a seguir.

Estilização no Head

Como mencionado no início, normalmente a estilização é realizada num arquivo separado, através do CSS. Porém, além da forma acima (diretamente no elemento que se quer estilizar), podemos fazer a estilização por classes, o que é feito no Head do projeto, com a tag style (note que aqui style é uma tag e não um atributo), como veremos a seguir.

Acima podemos notar que foi atribuída uma imagem a todo o corpo da página (body), através da tag style, situada no Head. Essa estilização pode ser feita apenas em uma classe determinada de elementos, por exemplo, os parágrafos, para isso precisamos criar essa classe através do atributo class, que vai identificar os elementos parágrafos que desejamos estilizar. Assim todo parágrafo que tiver com o atributo class com o mesmo valor (nome), receberá a mesma estilização.

Assim, podemos estilizar qualquer elemento do nosso HTML, sem precisar recorrer a recursos externos. Acima citamos apenas alguns exemplos. Se desejar saber mais, procure os links na página principal.
